# 03 - Landasan Teori

Folder ini berisi landasan teori, arsitektur penelitian, desain sistem, serta konsep-konsep yang digunakan sebagai dasar pelaksanaan penelitian **"Klasifikasi Penyakit Daun Padi Menggunakan EfficientNet-B6 dengan Pendekatan Transfer Learning"**.

Seluruh teori yang disajikan pada folder ini digunakan sebagai acuan dalam proses implementasi model, pelatihan, evaluasi, serta pembahasan hasil penelitian.

## Ringkasan

Folder **03-teori** menjadi dasar konseptual penelitian ini. Seluruh teori, desain penelitian, dan arsitektur sistem yang dijelaskan pada folder ini digunakan sebagai acuan dalam implementasi model EfficientNet-B6, mulai dari proses preprocessing dataset, pelatihan model menggunakan Transfer Learning, hingga evaluasi performa menggunakan Accuracy, Precision, Recall, F1-Score, Classification Report, dan Confusion Matrix.
