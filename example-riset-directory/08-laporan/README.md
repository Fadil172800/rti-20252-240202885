# 08 - Laporan Penelitian

Folder ini berisi laporan penelitian lengkap mengenai **Klasifikasi Penyakit Daun Padi Menggunakan EfficientNet-B6 dengan Pendekatan Transfer Learning**.

Laporan disusun berdasarkan seluruh tahapan penelitian, mulai dari perumusan masalah, studi literatur, pengolahan dataset, implementasi model, proses pelatihan, evaluasi, hingga pembahasan hasil penelitian.

Dokumen ini menjadi laporan utama yang mendokumentasikan keseluruhan proses penelitian secara sistematis.

---

## Tujuan Laporan

Laporan penelitian disusun untuk:

- Mendokumentasikan seluruh tahapan penelitian.
- Menjelaskan metodologi yang digunakan.
- Menyajikan hasil eksperimen secara lengkap.
- Menganalisis performa model EfficientNet-B6.
- Menjadi dasar penyusunan manuskrip ilmiah.

---

## Isi Laporan

Laporan penelitian terdiri atas beberapa bagian utama, yaitu:

1. Ringkasan Eksekutif
2. Latar Belakang dan Rumusan Masalah
3. Metodologi Penelitian
4. Hasil Penelitian
5. Perbandingan dengan Penelitian Acuan
6. Keterbatasan Penelitian
7. Saran Penelitian Lanjutan
8. Kesimpulan
9. Lampiran Artefak Penelitian
