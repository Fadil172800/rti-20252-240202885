# Evaluation Result

## Evaluasi Model

Evaluasi dilakukan menggunakan fungsi `model.evaluate()` pada dataset validasi yang berasal dari 20% data Rice Leafs.

Proses evaluasi menghasilkan nilai loss dan accuracy sebagai indikator performa model setelah proses pelatihan selesai.

---

## Hasil Evaluasi

| Metrik | Nilai |
|--------|-------:|
| Validation Loss | 1.0564 |
| Validation Accuracy | 63.04% |

---

## Ringkasan

Model EfficientNet-B6 memperoleh nilai **Validation Accuracy sebesar 63.04%** dengan **Validation Loss sebesar 1.0564** pada data validasi.

Hasil evaluasi ini menjadi dasar untuk analisis lanjutan menggunakan Confusion Matrix dan Classification Report guna mengetahui performa model pada setiap kelas penyakit daun padi.
