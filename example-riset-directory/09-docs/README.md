# 09 - Dokumentasi Penelitian

Folder ini berisi dokumentasi lengkap mengenai tahapan pelaksanaan penelitian **"Klasifikasi Penyakit Daun Padi Menggunakan EfficientNet-B6 dengan Pendekatan Transfer Learning"**.

Dokumen pada folder ini dibuat untuk mendokumentasikan seluruh proses penelitian mulai dari tahap perencanaan, persiapan dataset, implementasi model, pelatihan dan evaluasi model, hingga penyusunan manuskrip penelitian.

---

# Tujuan Dokumentasi

Dokumentasi penelitian disusun untuk:

- Mendokumentasikan setiap tahapan penelitian secara sistematis.
- Menjelaskan proses implementasi penelitian.
- Menjadi acuan selama proses penelitian berlangsung.
- Mempermudah reproduksi penelitian.
- Menjadi dokumentasi pendukung laporan dan manuskrip penelitian.

# Catatan

Folder **09-docs** merupakan pusat dokumentasi penelitian yang mencatat seluruh proses pelaksanaan penelitian dari awal hingga selesai.

Seluruh dokumen pada folder ini saling berkaitan dengan proposal, dataset, implementasi model, hasil eksperimen, manuskrip, dan laporan penelitian sehingga memudahkan peneliti maupun pembaca dalam memahami alur penelitian secara menyeluruh serta mereproduksi penelitian apabila diperlukan.
